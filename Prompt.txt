Constante Descrição
MYSQL_HOST
Endereço do servidor da base de dados. No caso de localhost, indica que o servidor corre na mesma
máquina.
MYSQL_DATABASE Nome da base de dados à qual se pretende aceder.
MYSQL_USERNAME Nome de utilizador com permissões para aceder à base de dados.
MYSQL_PASSWORD Palavra-passe do utilizador da base de dados.
MYSQL_AES_KEY Chave opcional para operações de encriptação, útil em sistemas que armazenam dados sensíveis.
